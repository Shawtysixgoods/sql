Создайте таблицу товаров интернет-магазина, заполните её данными и выполните запросы, связанные с числовыми значениями: ценой, количеством, скидкой, рейтингом и числом продаж. Для добавления записей используется `INSERT INTO`, а числовые выражения и агрегатные функции применяются прямо в запросах.

### Шаг 1. Создайте таблицу
Выполните готовый код:
```sql
CREATE TABLE products (
  product_id INTEGER PRIMARY KEY AUTOINCREMENT,
  product_name TEXT NOT NULL,
  category TEXT NOT NULL,
  price REAL NOT NULL,
  quantity INTEGER NOT NULL,
  discount_percent REAL NOT NULL,
  rating REAL NOT NULL,
  sold_count INTEGER NOT NULL
);
```

### Шаг 2. Заполните таблицу данными
Выполните готовый код:
```sql
INSERT INTO products (product_name, category, price, quantity, discount_percent, rating, sold_count) VALUES
('Ноутбук', 'Электроника', 75999.90, 12, 10, 4.8, 25),
('Мышь', 'Электроника', 1299.50, 40, 5, 4.4, 120),
('Клавиатура', 'Электроника', 3499.00, 25, 15, 4.6, 80),
('Стул офисный', 'Мебель', 8990.00, 8, 12, 4.3, 15),
('Стол письменный', 'Мебель', 15400.00, 5, 7, 4.7, 10),
('Лампа настольная', 'Освещение', 2300.00, 18, 20, 4.5, 55);
```
SQLite позволяет вставлять несколько строк одной командой `INSERT INTO ... VALUES (...)`. 

### Шаг 3. Проверьте содержимое таблицы
```sql
SELECT * FROM products;
```

## Вычисления

В этих заданиях нужно применить арифметические операции. SQLite поддерживает вычисления в выражениях запроса и встроенные функции для обработки чисел. 

### 11. Выведите название товара и цену, увеличенную на 12%
Подсказки:
- Используйте умножение.
- Новую цену можно посчитать через исходную цену и процент увеличения.
- Задайте псевдоним, например `new_price`.

### 12. Выведите название товара и сумму выручки, если проданы все единицы из `sold_count`
Подсказки:
- Нужно умножить `price` на `sold_count`.
- Выведите также `product_name`.

### 13. Округлите цену до целого значения
Подсказки:
- Используйте функцию округления.
- Передайте в неё поле `price`. 

### 14. Найдите товары, у которых скидка от 10 до 20 процентов
Подсказки:
- Используйте два условия сравнения.
- Объедините их через `AND`.

### 15. Найдите товары категории `Электроника`, у которых рейтинг выше 4.5
Подсказки:
- Нужно условие и по текстовому, и по числовому полю.
- Используйте `WHERE` и `AND`.
