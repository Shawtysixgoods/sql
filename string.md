Создайте таблицу со студентами, заполните её данными и выполните запросы на обработку строк. В SQLite таблица создается через `CREATE TABLE`, данные добавляются через `INSERT INTO`, а строки можно обрабатывать встроенными функциями и оператором `||`. 

## Часть 1: с кодом

### Шаг 1. Создайте таблицу
Напишите и выполните следующий код:
```sql
CREATE TABLE students (
  student_id INTEGER PRIMARY KEY AUTOINCREMENT,
  last_name  TEXT NOT NULL,
  first_name TEXT NOT NULL,
  middle_name TEXT,
  group_code TEXT NOT NULL,
  email      TEXT UNIQUE,
  city       TEXT
);
```
В таблице используются ограничения `PRIMARY KEY`, `NOT NULL` и `UNIQUE`, которые SQLite поддерживает при создании таблиц. 

### Шаг 2. Добавьте данные
Выполните готовый код:
```sql
INSERT INTO students (last_name, first_name, middle_name, group_code, email, city) VALUES

```
SQLite позволяет вставлять несколько строк одной командой `INSERT INTO ... VALUES (...)`.

## Часть 2: допишите сами

Ниже даны задания и подсказки. Код студенты должны написать самостоятельно.

### 1. Выведите фамилию и имя в одном столбце
Подсказки:
- Используйте `SELECT`.
- Объедините `last_name` и `first_name`.
- Между ними добавьте пробел.
- Для объединения строк в SQLite используется `||`. 

Ожидаемое имя столбца: `full_name`.

### 2. Выведите фамилии в верхнем регистре
Подсказки:
- Используйте функцию `UPPER()`. 
- Внутрь функции передайте поле с фамилией.
- Задайте псевдоним столбцу, например `last_name_upper`.

### 3. Выведите имена в нижнем регистре
Подсказки:
- Используйте функцию `LOWER()`. 
- Работайте со столбцом `first_name`.

### 4. Найдите длину фамилии каждого студента
Подсказки:
- Используйте функцию `LENGTH()`. 
- Выведите саму фамилию и её длину.
- Подумайте, какое имя дать вычисляемому столбцу.

### 5. Выведите первые 3 буквы фамилии
Подсказки:
- Используйте `SUBSTR(строка, начало, длина)`.
- Начальная позиция в SQLite задается с 1.
- Нужно взять поле `last_name`, начиная с 1 символа, длиной 3 символа.

### 6. Найдите студентов, у которых город начинается с буквы «С»
Подсказки:
- Используйте `WHERE`.
- Для поиска по шаблону используйте `LIKE`. 
- Символ `%` означает любую последовательность символов. 
- Шаблон должен показывать: строка начинается на букву `С`.

### 7. Найдите студентов, у которых в email есть `mail`
Подсказки:
- Используйте `WHERE email LIKE ...`
- Подумайте, где нужно поставить `%`, чтобы искать фрагмент внутри строки.

### 8. Сформируйте строку вида `Студент: Иванов Иван`
Подсказки:
- Объедините текстовую константу и поля таблицы.
- Используйте `||`. 
- Не забудьте добавить пробелы между словами.

### 9. Выведите студентов, у которых имя длиннее 5 символов
Подсказки:
- В условии `WHERE` можно использовать `LENGTH()`. 
- Сравните длину имени с числом 5.
